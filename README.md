Este repositório contém o código-fonte do meu portfólio pessoal, desenvolvido com o objetivo de apresentar minha trajetória, habilidades e projetos como desenvolvedor Back-end.

O projeto foi construído utilizando HTML, CSS e Bootstrap, priorizando uma interface moderna, responsiva e de fácil navegação.

---

## 🚀 Funcionalidades

* Página inicial com apresentação pessoal.
* Seção "Sobre" com minha trajetória profissional.
* Listagem de habilidades e tecnologias.
* Exibição de projetos desenvolvidos.
* Área de contato com links para GitHub, LinkedIn e e-mail.
* Layout responsivo para diferentes dispositivos.

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* Bootstrap 5
* Bootstrap Icons
* Google Fonts
